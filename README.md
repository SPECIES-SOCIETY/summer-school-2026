# summer-school-2026
Github repository for the SPECIES Summer School 2026

## GitHub Pages

This repository includes a GitHub Pages site. Once enabled in the repository settings, the site will be available at:
`https://species-society.github.io/summer-school-2026/`

To enable GitHub Pages:
1. Go to the repository Settings
2. Navigate to "Pages" in the left sidebar
3. Under "Build and deployment", select "Deploy from a branch"
4. Select the branch (e.g., `main` or your current branch) and the root folder `/`
5. Click Save

The site will be built automatically using Jekyll.
